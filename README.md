## Simple Description (간단한 설명)

<details>
<summary>English</summary>
A modern abstract strategy board game inspired by chess and janggi. 
Featuring spy units, territory control, and dual victory conditions (intel vs commander).
</details>

<details>
<summary>Korean</summary>
체스와 장기에서 영향을 받은 현대적인 추상 전략 보드게임입니다.
첩자, 영토, 대사관 시스템을 중심으로 정보전 승리 루트와 수반 기물을 잡는 루트 두 가지 승리 루트를 지원하는 것이 특징입니다.
</details>
